- Bilder unter src/pic ablegen.
- Latex-Quellen für weitere Kapitel unter src/tex ablegen.
- PDF erzeugen durch 'make' oder 'make all', aufräumen durch make clean.
- Ausgabeordner target in CVS/SVN/GIT ignore setzen!
- make kann unter Windows bspw. mit cygwin genutzt werden (http://www.cygwin.com/).
- zur Ausfürhung wird eine Tex Umgebung benötigt, z.B. MikTek (http://miktex.org/).
- Werden Dateien umbenannt, muss ggf. das makefile angepasst werden.
